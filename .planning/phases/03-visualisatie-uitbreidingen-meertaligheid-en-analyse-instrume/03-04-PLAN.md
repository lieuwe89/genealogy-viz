Create `public/js/clustering.js` — the `window.Clustering` IIFE module.

Module state:
- `clusterLevel`: `'persons'` | `'surname'` | `'century'` (current level)
- `clusterLocked`: `false` (manual lock state per D-04)
- `rawGraphData`: `null` (original unmodified graph data, stored on init)
- `THRESHOLD_CENTURY`: `1200` (camera distance for century level)
- `THRESHOLD_SURNAME`: `500` (camera distance for surname level)

Exported functions:

**`init(graphData)`** — Store a deep reference to the original graphData. Called once from initViz.

**`buildClusteredData(level, graphData, surnamePalette)`** — Returns `{ nodes, links }`:
- If `level === 'persons'`: return original graphData unchanged
- If `level === 'century'`: group nodes by century (Math.floor(birthYear / 100) * 100). Nodes without birthYear go into group `'unknown'`. For each group, create a synthetic cluster node:
  ```javascript
  {
    id: '__cluster_century_' + century,   // Pitfall 2: non-numeric prefix
    name: century === 'unknown' ? '?' : String(century) + 's',
    _isCluster: true,
    _level: 'century',
    _members: memberIds,                  // array of real node IDs
    _radius: Math.max(8, Math.sqrt(members.length) * 8),
    _color: '#1e3a5f',                    // per UI-SPEC: century plane colour
    _labelColor: 'rgba(100,160,255,0.7)', // per UI-SPEC
    x: avg(members, 'x'),
    y: avg(members, 'y'),
    z: avg(members, 'z'),
  }
  ```
  Return `{ nodes: clusterNodes, links: [] }` (no links between century clusters).

- If `level === 'surname'`: group nodes first by century, then within each century by surname. Nodes without surname go into `'Onbekend'` group (per Claude's Discretion). For each surname group, create a synthetic node:
  ```javascript
  {
    id: '__cluster_surname_' + century + '_' + surname,
    name: surname || 'Onbekend',
    _isCluster: true,
    _level: 'surname',
    _members: memberIds,
    _radius: Math.max(5, Math.sqrt(members.length) * 5),
    _color: surnamePalette[surname] || '#6e7681',  // per UI-SPEC: surname palette at 20% opacity in render
    _labelColor: 'rgba(255,255,255,0.6)',           // per UI-SPEC
    x: avg(members, 'x'),
    y: avg(members, 'y'),
    z: avg(members, 'z'),
  }
  ```
  Return `{ nodes: surnameClusterNodes, links: [] }`.

**`getAutoLevel(cameraDistance)`** — Returns level string based on thresholds:
- `cameraDistance > THRESHOLD_CENTURY` → `'century'`
- `cameraDistance > THRESHOLD_SURNAME` → `'surname'`
- else → `'persons'`

**`setLevel(level)`** — Sets clusterLevel, returns the new level.

**`setLocked(locked)`** — Sets clusterLocked.

**`isLocked()`** — Returns clusterLocked.

**`getLevel()`** — Returns current clusterLevel.

**`finerLevel(currentLevel)`** — Returns the next finer level: `'century'` → `'surname'`, `'surname'` → `'persons'`, `'persons'` → `'persons'`.

Helper: `function avg(nodes, prop)` — average of a property across nodes, defaulting to 0 for missing values.

```javascript
// Auto-clustering based on camera distance (per D-01)
let clusterDebounceTimer = null;
function onCameraChange() {
  if (clusterDebounceTimer) clearTimeout(clusterDebounceTimer);
  clusterDebounceTimer = setTimeout(function() {
    if (!window.Clustering || Clustering.isLocked()) return;
    var dist = graph.camera().position.length();
    var newLevel = Clustering.getAutoLevel(dist);
    if (newLevel !== currentClusterLevel) {
      applyClusterLevel(newLevel);
    }
  }, 200);  // 200ms debounce per RESEARCH.md Pitfall 3
}

// Try graph.controls().addEventListener first; fall back to requestAnimationFrame polling
try {
  var controls = graph.controls();
  controls.addEventListener('change', onCameraChange);
} catch (e) {
  // Fallback: poll camera distance (Assumption A1 fallback)
  (function pollCamera() {
    onCameraChange();
    requestAnimationFrame(pollCamera);
  })();
}
```
