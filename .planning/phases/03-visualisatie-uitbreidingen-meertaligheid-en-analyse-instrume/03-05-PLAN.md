What was built:
- i18n: Language toggle (EN/NL) in toolbar; all UI labels translate between Dutch and English
- Clustering: Zoom-based node clustering (century, surname, individual levels); cluster select + lock in toolbar; click cluster to expand
- Relationship path: Named relationship label in two-person comparison view (below timeline)

How to verify:

i18n verification:
1. Open http://localhost:3000/genealogy-viz/app.html
2. Verify toolbar buttons show Dutch labels (Op rol, Op achternaam, Op geslacht, Overlap)
3. Click "EN" button — labels switch to English (By Role, By Surname, By Sex, Overlap)
4. Reload page — language persists as English
5. Click "NL" — back to Dutch
6. Open http://localhost:3000/genealogy-viz/admin/ — verify admin labels are in current language
7. Click language toggle in admin — verify labels switch

Clustering verification:
8. Return to app.html, zoom out far using mouse wheel — nodes should collapse into large translucent century bubbles with year labels
9. Zoom to medium distance — surname clusters should appear
10. Zoom in fully — individual person nodes visible
11. Click a century cluster bubble — camera should fly into it, showing surname clusters
12. Use the cluster level dropdown in toolbar to manually select "Eeuwen" (Centuries)
13. Click "Clusters: auto" button — it should become "Clusters: vast" (locked), zooming should no longer change the cluster level
14. Click again to unlock

Relationship path verification:
15. Click any person node — side panel opens
16. In the Connections section, click "Tijdlijn" next to a connected person
17. Two-person comparison view shows: timeline bars AND below them a "RELATIE" section with a named label (e.g., "Voorouder (1 generaties)" or "Echtgeno(o)t(e)")
18. Switch language to EN — relationship label should show in English (e.g., "Ancestor (1 generations)")

No errors:
19. Open browser DevTools Console — verify no JavaScript errors throughout all interactions
