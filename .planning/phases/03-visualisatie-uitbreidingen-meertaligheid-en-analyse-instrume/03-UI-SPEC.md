## Interaction Contracts

### Toolbar: Language Toggle (D-06)
- Button follows existing `.toolbar-btn` style.
- Placement: immediately after the existing colour mode and overlap buttons (rightmost before the admin badge area).
- Active state: `.toolbar-btn.active` styling when Dutch is selected (default active).
- On click: swap language, update `localStorage.setItem('lang', 'nl'|'en')`, re-render all i18n-keyed strings.
- Label shows the language you will switch TO (convention: `EN` when currently Dutch, `NL` when currently English).

### Toolbar: Cluster Level Controls (D-01, D-04)
- A compact `<select>` or segmented button group — use `<select>` styled to match `.edit-field select` pattern.
- Width: 120px max. Placed in toolbar immediately after language toggle.
- Three options: `Eeuwen`, `Geslachtsnamen`, `Personen` (i18n keyed).
- A lock toggle button (`.toolbar-btn`) beside the select: when active (`.toolbar-btn.active`), zoom changes do not override the selected level.
- When unlocked (default), the select reflects the auto-detected level based on current camera distance — it updates as the user zooms.

### 3D Scene: Cluster Bubbles (D-02, D-03)
- Cluster bubbles are THREE.js `Mesh` objects with `SphereGeometry` or `Sprite`, rendered in the scene alongside individual nodes.
- At century level: large translucent sphere (`opacity: 0.12`) using century plane colour `#1e3a5f`. Label sprite above the sphere.
- At surname level: medium sphere using the surname's palette colour at 20% opacity. Label sprite above the sphere.
- Click on a cluster bubble: zoom camera into that cluster and switch cluster level one step finer (century → surname → persons). Animate over 600ms using existing OrbitControls.
- Cluster label font: 28px monospace on a 256×64 canvas (matches existing century year label pattern in `viz.js`).
- Cluster label colour: `rgba(100,160,255,0.7)` for century clusters; `rgba(255,255,255,0.6)` for surname clusters.

### Side Panel: Relationship Label (D-10, D-11, D-13)
- Rendered inside `renderTwoPersonMode()` in `side-panel.js`, below the existing `.timeline-wrap` section.
- Container: a new `.panel-section` div (matches existing pattern with `border-bottom: 1px solid #21262d`).
- Structure: `.panel-label` uppercase header ("RELATIE" / "RELATIONSHIP") at 11px with `letter-spacing: 0.08em`, then a single `<p>` with the named label text at 14px (body role).
- If no path exists: show "Geen verbinding gevonden" in `#6e7681` muted colour.
- If path analysis errors: show "Kon relatie niet bepalen" in `#6e7681`.

### i18n Runtime Behaviour (D-05, D-07, D-08)
- `window.i18n` module exposes `t(key)` function.
- On language switch: call `i18n.setLang('nl'|'en')`, then re-render all dynamic HTML sections (toolbar button titles, panel sections, admin headings).
- Static HTML attributes (`title`, `placeholder`, `aria-label`) updated via `document.querySelectorAll('[data-i18n]')`.
- Admin panel: uses `data-i18n` attributes on headings, labels, nav items — updated on load and on language switch.
